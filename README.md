# 패키지 설치

<b>bosun_repo 클론</b>
```
git clone https://github.com/aris-hi5/bosun_repo.git
```

<b>bosun_repo 레포지토리 이동</b>
```
cd bosun_repo
cd hi5_prj
```
<b>xarm-Python-SDK 클론</b>
```
git clone https://github.com/xArm-Developer/xArm-Python-SDK.git<br>
```

<b>xarm-Python-SDK 패키지 설치</b>
```
cd xarm-Python-SDK
pip install .
```
